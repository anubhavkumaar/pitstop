# Pit Stop — Freelance Auto Services

> GTA V Roleplay · San Andreas · Mechanic & Detailing Crew

## Overview

Pit Stop is a freelance auto services crew operating out of the SouthSide Car Wash in Los Santos. The website covers service bookings, job logs, and crew information for the roleplay division.

**Live site:** [pitstop-services.web.app](https://pitstop-services.web.app)

## Tech Stack

- React 19 + Vite 8
- Framer Motion 12
- React Router DOM 7
- Firebase (Firestore + Hosting)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Deploy

```bash
npm run deploy
```

Builds and pushes to Firebase Hosting automatically. Requires `firebase-tools` installed globally and authenticated.

> CI/CD: every push to `main` triggers a build and deploy via GitHub Actions.
